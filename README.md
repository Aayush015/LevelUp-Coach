# LevelUp: Adaptive Endurance Engine
**A data-driven, life-aware training orchestrator for marathon and triathlon athletes.**

LevelUp Coach is a full-stack platform that solves the "static plan" problem. It aggregates biometric and activity data to create a dynamic feedback loop that adjusts training intensity based on real-world recovery and professional schedules.

---

## 🚀 The Core Problem
Traditional training plans are rigid. They don't account for a 10-hour work shift, poor sleep, or high-stress days. LevelUp replaces "one-size-fits-all" PDFs with an **Adaptive Logic Engine** that prioritizes injury prevention and realistic goal setting.

## ✨ Key Features
* **Unified Data Fusion:** Normalizes disparate data from **Apple Health** (Sleep, HRV) and **Strava** (GPS, Pace) into a single "Source of Truth."
* **Work-Life Integration:** A "High Stress" toggle that automatically down-regulates intensity during heavy professional periods.
* **Predictive Readiness:** Calculates **Chronic Training Load (CTL)** to validate if a goal (e.g., an Ironman) is realistic based on current volume.
* **Smart Deduplication:** Proprietary logic to merge overlapping activities from multiple sources without doubling volume.
